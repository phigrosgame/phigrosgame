# Phigros 粉絲網站

一個 Phigros 音樂遊戲的粉絲資訊網站，包含遊戲介紹、歌曲列表等內容。

## 網站特色

- 🎮 **遊戲介紹** - 深入了解 Phigros 的獨特玩法與魅力
- 🎵 **歌曲列表** - 瀏覽遊戲中的熱門歌曲，依難度篩選
- 📱 **響應式設計** - 支援手機、平板、電腦各種裝置
- ✨ **動畫效果** - 流暢的頁面動畫與互動效果
- 🌙 **深色主題** - 霓虹風格的深色介面設計

## 網站頁面

1. **首頁** - 遊戲特色與預覽
2. **遊戲介紹** - 玩法、難度、評分系統說明
3. **歌曲列表** - 熱門歌曲展示與難度篩選
4. **關於我們** - 網站介紹與聯絡資訊
